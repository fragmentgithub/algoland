# アルゴランド 🤖

4歳児向けのアルゴリズム学習ミニゲーム集。ブラウザだけで動きます(インストール・ビルド不要)。

## あそびかた

`index.html` をダブルクリックしてブラウザで開くだけ。タブレットのタッチ操作にも対応(ドラッグなし・すべてタップ)。

## ゲームとねらい

| ゲーム | あそびかた | 育つ力(アルゴリズム) |
|---|---|---|
| 🚂 すうじのきしゃ | 散らばった数字を 1 から順にタッチして機関車に連結 | 数の順序・逐次処理 |
| 🦒 せいくらべ | 「いちばん小さいのはどれ?」を繰り返して整列(むずかしいモードは大きい順も) | 比較と並び替え(選択ソート) |
| 🍎 つぎは なあに? | 🍎🍌🍎🍌…の「つぎ」を 3 択から選ぶ(AB→ABC→AABB と難化) | 規則性・パターン認識 |
| 🤖 ロボットの おつかい | ⬆️➡️ を並べてから GO!(むずかしいモードは岩をよける) | 手順の計画→実行→デバッグ |
| 🧺 なかまわけ | たべもの・のりもの・いきものをバスケットに分ける | 分類 |
| 🔍 かたちさがし | 「あかい まるは どれ?」を条件で探す | 条件判定(AND) |

## 機能

- **難易度 3 段階**(かんたん/ふつう/むずかしい)— ⚙️設定から変更
- **1 日のプレイ時間制限** — 初期値 15 分(5/10/15/20/30 分/制限なしを選択可)。時間が来ると「おやすみ画面」になり、**暗証番号**(初期値 `1234`)でリセット。日付が変わると自動リセット
- **⚙️設定は暗証番号ゲート付き**(難易度・時間・声の速さ・暗証番号変更)
- **おともだちコレクション** — ⭐がたまるごとに 9 匹の仲間が増える(3/6/10/15/20/26/33/40/50 個)。25 個で🌈、50 個で🏰がホームに出現
- **音声読み上げ** — 日本語ボイスを自動選択(Edge の Nanami 等)。指示・ほめ言葉・ヒントを読み上げ。速度は「ゆっくり/ふつう」切替可
- 2 回間違えると正解が光るヒント、効果音は Web Audio 生成(音声ファイルなし)

## 技術メモ

- 単一の `index.html`(Vanilla JS / 依存なし)
- 進捗・設定は localStorage(`algo4.*`)に保存
- `prefers-reduced-motion` 対応
- 非表示タブではプレイ時間をカウントしない
